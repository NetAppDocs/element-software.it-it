= Scopri come impostare l'archiviazione
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
A questo punto, l'hardware dovrebbe essere installato. L'hardware include anche software Element.

Successivamente, è necessario configurare il sistema di storage per il proprio ambiente. È possibile configurare un cluster con nodi di storage o Fibre Channel e gestirlo utilizzando il software Element dopo l'installazione e l'accensione dei nodi in un'unità rack.

image::../media/sf_and_element_workflow_for_setup_shorter_workflow.png[Diagramma del flusso di lavoro di installazione]

.Procedura per la configurazione dello storage
. Selezionare una delle seguenti opzioni:
+
** link:../setup/concept_setup_configure_a_storage_node.html["Configurare il cluster con nodi di storage"]
+
È possibile configurare un cluster con nodi di storage e gestirlo utilizzando il software Element dopo aver installato e collegato i nodi in un'unità rack e averli accesi. È quindi possibile installare e configurare componenti aggiuntivi nel sistema di storage.

** link:../setup/concept_setup_fc_configure_a_fibre_channel_node.html["Configurare il cluster con nodi Fibre Channel"]
+
È possibile configurare un cluster con nodi Fibre Channel e gestirlo tramite il software Element dopo aver installato e cablato i nodi in un'unità rack e averli accesi. È quindi possibile installare e configurare componenti aggiuntivi nel sistema di archiviazione.
